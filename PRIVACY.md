# Privacy Policy

This GPT uses an external API to store and retrieve user-provided memory.

## Data Collected
- User-provided text explicitly requested to be remembered.
- No passwords, authentication secrets, or financial credentials are stored.

## Data Usage
- Data is used only to provide memory recall functionality to the user.
- Data is not shared with third parties.

## Data Storage
- Data is stored on a private server controlled by the developer.
- Data persists until explicitly deleted by the user.

## Contact
For questions or deletion requests, contact the developer via GitHub.
